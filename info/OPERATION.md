| Icon | Function |

|--------------------------------------------------|------------------------------------------------------------|

<details><summary><h2>Context Menu</h2></summary>
Depending on the selected point of the tour, the following functions are available:

| Function | Description |

|------------------------------------|----------------------------------------|

| Break Time | Individual break time per waypoint |

| Comment on waypoint | Comment in travel time table |

| Navigate to waypoint | Transfer to external map apps |

| To the Black Forest Association Tour Portal | Displays a map with suggested routes |

Google Navigation | Navigation with Google Maps |

OpenStreetMap POIs nearby | Add POIs within 150 m |

Irreversible: |

**Start from here** | |

**Remove waypoint** | |

**Remove all points after this point** | |

</details>
